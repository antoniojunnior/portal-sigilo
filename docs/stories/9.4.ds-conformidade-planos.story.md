### `cn()` — NÃO disponível neste projeto

A spec de Uma (seção 4.4) usa `cn(...)` para classes condicionais no card wrapper.
**`clsx` e `tailwind-merge` não estão no `package.json` deste projeto.**

Usar o padrão do projeto para classes condicionais (mesmo do `Button.tsx`):

```tsx
// ✅ Padrão do projeto
className={[
  "relative flex flex-col rounded-2xl border p-6 transition-shadow",
  "shadow-[var(--shadow-sm)] hover:shadow-[var(--shadow-md)]",
  plano.destaque
    ? "border-2 border-[var(--color-primary)] bg-[var(--color-primary-surface)]"
    : "border border-[var(--color-border)] bg-[var(--color-card)]",
].join(" ")}

// ❌ NÃO fazer (cn não existe)
className={cn("...", plano.destaque ? "..." : "...")}
```
