## Story

**As a** visitante em `portalsigilo.com.br/planos`,
**I want** visualizar uma página clara com os 3 planos disponíveis e clicar em "Contratar" para ser redirecionado ao checkout Asaas,
**so that** eu possa contratar o Portal Sigilo sem precisar entrar em contato manual com a equipe de vendas.

## Acceptance Criteria

1. **Página `/planos` funcional**
   - Tabela ou cards comparativos com os 3 planos: Entrada, Gestão, Enterprise
   - Exibe para cada plano: preço mensal e anual, lista de features, limites (usuários, armazenamento, canais)
   - Features indisponíveis exibidas com visual desabilitado (ícone cinza ou riscado)
   - Dados de preço e features extraídos do PRD seção 3 — não hardcoded em múltiplos lugares

2. **Botão "Contratar" — Entrada e Gestão**
   - Clique chama `POST /api/checkout/create` com `{ plano: 'entrada' | 'gestao' }`
   - Route Handler server-side cria link de cobrança na API Asaas e retorna `{ url }`
   - Browser redireciona para a URL Asaas retornada
   - Loading state no botão durante a chamada
   - Erro tratado: exibir toast de erro se a API falhar

3. **Botão "Falar com vendas" — Enterprise**
   - Redireciona para WhatsApp ou e-mail de contato (configurável via variável de ambiente `NEXT_PUBLIC_SALES_CONTACT`)
   - Não aciona nenhuma chamada de API

4. **Segurança**
   - `ASAAS_API_KEY` presente APENAS em `src/app/api/checkout/create/route.ts` (server-side)
   - Nenhum dado de pagamento passa pelo Next.js — apenas redirect para URL Asaas
   - Route Handler valida que `plano` é um dos valores permitidos antes de chamar Asaas

5. **Compatibilidade e layout**
   - Página responsiva (320px a 1440px)
   - Segue design system existente do dashboard (Tailwind + shadcn/ui)
   - Não quebra nenhuma rota existente

```typescript
// POST /v3/paymentLinks — Asaas API
// Headers: { access_token: ASAAS_API_KEY }
// Body:
{
  name: "Portal Sigilo — Plano Entrada",
  billingType: "CREDIT_CARD",
  chargeType: "RECURRENT",
  cycle: "MONTHLY",
  value: 117.00, // ou 97.00 para anual
  description: "Assinatura Portal Sigilo Plano Entrada"
}
// Response: { url: "https://www.asaas.com/c/xxxxx" }
```

Sandbox URL: `https://sandbox.asaas.com/api/v3/`

Selecionar via `ASAAS_SANDBOX` em env.ts.
